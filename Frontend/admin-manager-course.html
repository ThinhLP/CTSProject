Thông tin chung

Java OOP

Java là ngôn ngữ lập trình sử dụng phổ biến nhất thế giới. Java có thể được dùng để xây dựng nhiều ứng dụng khác nhau như viết các ứng dụng web, mạng xã hội; xây dựng các ứng dụng quản lý trong doanh nghiệp, siêu thị, bệnh viện, ngân hàng...Những điều bạn được học trong khóa Java căn bản: Tìm hiểu kiến thức cơ bản của Java và cách thức lập trình hướng đối tượng của Java. Tìm hiểu kiểu dữ liệu chuỗi trong Java và các biểu thức có quy tắc, thường gọi là Regular Expression. Kỹ thuật xử lý lỗi phát sinh trong quá trình thực thi chương trình, gọi là Exception Handling. Tìm hiểu các loại dữ liệu trong bộ Collection Framework của Java. Khóa học được xây dựng dựng trên mô hình Project-based, có nghĩa bạn sẽ xây dựng project xuyên suốt trong khóa học. Khóa học bao gồm 4 phần chính, cũng tương đương bạn sẽ xây dựng một ứng dụng được nâng cấp phiên bản 4 lần.
